# v3.15.0 发布说明

**发布日期：** 2026-05-25

---

## 新功能

### Hook Machine — 三模型自动切换瀑布（#hooks）
新增 Groq → Gemini → GitHub Models 自动故障转移机制，任意一个供应商不可用时自动切换至下一个，彻底解决单点 AI 配额超限导致服务中断的问题。

- **Groq**（llama-3.3-70b-versatile）：主力，通常 < 3s 响应
- **Gemini 2.5 Flash**：备用，限时 18s
- **GitHub Models / gpt-4o-mini**：兜底，限时 12s
- 三路全失才返回通用 502 错误，不再暴露原始供应商错误信息
- `maxDuration` 提升至 60s（Groq 8s + Gemini 18s + GitHub 12s = 38s 最坏情况）

### 财务仪表盘 — 可点击 KPI 卡片 + 详情抽屉（#finance）
订阅数、AI 成本等 KPI 卡片支持点击，右侧滑出详情抽屉，显示具体用户列表或每日消耗明细。

- 新增 `/api/admin/finance/details` 端点（支持 `type=subscription|finops`）
- `AdminKpiCard` 新增 `onClick` prop，悬停时呈现高亮边框

### SEO — Bing Webmaster 验证 + IndexNow 修复（#seo）
- 在 `<head>` 直接注入 `msvalidate.01` 验证标签（替换不生效的 `metadata.verification.other` 方案）
- Bing 站点地图提交接口从废弃的 `/ping` 端点（HTTP 410）迁移至正式 Webmaster API
- IndexNow 改为 POST urlList 方式提交，新增 `/api/indexnow-verify` 自动服务密钥验证文件

## 升级说明

需在生产 Vercel 项目（`feng2027/ttlike`）Environment Variables 中确保以下三个 Key 均已设置：

```
GROQ_API_KEY      — console.groq.com（免费）
GITHUB_TOKEN      — github.com/settings/tokens（Classic，无需额外权限）
GEMINI_API_KEY    — aistudio.google.com/apikey
```

设置完毕后 Redeploy 即可激活三模型自动切换。
